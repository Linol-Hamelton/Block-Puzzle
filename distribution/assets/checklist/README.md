# Store Assets Checklist

Use this folder to keep source exports for store submissions.

## 1. Mandatory
1. App icon source (`1024x1024`, no rounded corners baked in)
2. Feature graphic / promo cover
3. Phone screenshots (portrait)
4. Tablet screenshots
5. Optional promo video link

## 2. Naming Convention
1. `icon_vNN.png`
2. `feature_graphic_vNN.png`
3. `screenshot_phone_01_vNN.png`
4. `screenshot_tablet_01_vNN.png`

## 3. Quality Rules
1. Use real gameplay screens, not mockups only.
2. Keep text overlays minimal and readable.
3. Do not claim unavailable features in visuals.
