# Brand Foundation v1 (Current)

## 1. Brand Core
- Product name: `Lumina Blocks`
- Current store title baseline:
  - RU: `Lumina Blocks: Дзен Пазл`
  - EN: `Lumina Blocks: Puzzle Flow`
- Category: Casual Puzzle
- Positioning: premium visual puzzle flow with clean controls and clear feedback.

## 2. Value Proposition
- Fast onboarding, deep score mastery.
- Satisfying visual rhythm (glow, clarity, combo feedback).
- Ad-free-first player experience with optional IAP utility/cosmetic progression.

## 3. Brand Personality
- Clear
- Modern
- Focused
- Rewarding
- Calm

## 4. Visual Direction
- Dark premium base (deep blue/navy gradients).
- Neon glass accents (cyan/violet/amber) aligned with block visuals.
- Clean geometry and high readability for gameplay-first UX.
- Motion tied to outcomes (line clear, combo, milestones), no noisy idle animation.

## 5. Voice and Tone
- Short and direct.
- Positive but not noisy.
- No aggressive monetization language.

## 6. Trademark and Naming Notes
- Primary naming route for launch: `Lumina Blocks`.
- Keep prior temporary names (`Block Puzzle:*`) as historical drafts only.

## 7. Required Brand Deliverables
1. App icon set (adaptive + legacy + web/favicon)
2. Splash/logo assets (dark and light variants)
3. Store feature graphic
4. Phone/tablet screenshots
5. Localized metadata (RU + EN)
6. Privacy policy URL + support email
